Spray & Wash Operations V4.0.22

Purpose:
Stabilisation/correction release after V4.0.21. This package applies a dedicated V4.0.22 front-end patch to complete missing Height dashboard, Equipment, Certificate, Reports and Account UI items.

Install:
1. Replace index.html, operations-v4.js and service-worker.js in the GitHub repo root.
2. Do not replace config.js.
3. No Supabase SQL is required.
4. Open https://sprayandwash.github.io/height-safety-register/?v=4.0.22&fresh=1 and hard refresh.

If the app appears cached, clear site data in DevTools > Application > Storage.
