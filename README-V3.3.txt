Spray & Wash Height Safety Register V3.3

In-app notifications release.

What changed:
- Notification bell in the top-right of the app
- Notification centre inside the app
- Dashboard notification summary
- Configurable due-soon lead time: 7, 14, 30, 60 or 90 days
- Alerts for failed/quarantined items
- Alerts for overdue inspections
- Alerts for items due soon
- Alerts for active items with no inspection history
- Alerts for active items with no equipment photos
- Alerts where the latest inspection has no inspection photos
- Notification tiles link to filtered equipment lists

No Supabase SQL is required. This is an app/interface update only.

Upload/replace these files in GitHub:
- index.html
- app.js
- manifest.webmanifest
- service-worker.js
- assets/icon.svg

Do not replace config.js.
