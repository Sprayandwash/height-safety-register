Spray & Wash Height Safety Register V3.2

Includes:
- Header/version patch: Version 3.2
- Service worker cache refresh
- Certificates tab
- Generate certificates for selected items
- Generate certificates for all equipment of selected type
- Generate certificates for inspections within a date range
- Generate certificates by inspection result
- Generate certificates for due/overdue items
- Certificate history table
- Regenerate certificates from history
- Certificate CSV export

Install:
1. Run supabase-schema-v3.2.sql in Supabase SQL Editor.
2. Upload/replace these files in GitHub root:
   - index.html
   - app.js
   - manifest.webmanifest
   - service-worker.js
   - assets/icon.svg
3. Also upload supabase-schema-v3.2.sql and README-V3.2.txt for records.
4. Do not replace config.js.

Certificates open in a printable browser page. Use Print > Save as PDF to create a PDF.
