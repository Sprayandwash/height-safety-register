V4.0.9 Release Notes

- Standardised look and feel across modules using the coloured Height Equipment dashboard card style.
- Made the app logo clickable to return to the main Home dashboard.
- Standardised Home navigation button styling.
- Simplified the Admin module:
  - Removed redundant Admin tabs.
