Spray & Wash Height Safety Register - GitHub Pages Publish Package

Upload these files/folders to the root of your GitHub repository:

- index.html
- manifest.webmanifest
- service-worker.js
- README.txt
- icons/
- assets/

GitHub Pages setup:
1. Create a new repository called height-safety-register.
2. Upload the extracted contents of this ZIP to the repository root.
3. Go to Settings > Pages.
4. Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Click Save.
6. Wait 1-2 minutes.
7. Your app URL will be:
   https://YOUR-GITHUB-USERNAME.github.io/height-safety-register/

Install:
- iPhone: open the URL in Safari > Share > Add to Home Screen.
- Android: open the URL in Chrome > menu > Install app or Add to Home screen.

Important:
- App data is stored locally on each device/browser.
- Use Export JSON regularly for backup.
