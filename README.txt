Spray & Wash Height Safety Register - Supabase Cloud Version

Setup:
1. Create a Supabase project.
2. Run supabase-schema.sql in Supabase SQL Editor.
3. Enable Email auth in Authentication > Providers.
4. Copy Project URL and anon public key from Project Settings > API.
5. Paste them into config.js.
6. Upload these files to GitHub Pages.
7. Open the app and create/sign in with a user account.

This version stores data in Supabase so phone and laptop share the same register.
