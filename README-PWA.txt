Spray & Wash Height Safety Register - Cloud PWA

Upload these files to the root of your GitHub repository, replacing the current files.

Important:
1. Keep your working Supabase values in config.js.
2. If this package contains placeholder values in config.js, copy your current working config.js values into it before uploading.
3. Upload:
   - index.html
   - app.js
   - config.js
   - manifest.webmanifest
   - service-worker.js
   - assets/
4. Wait 1-2 minutes for GitHub Pages.
5. On Android Chrome, open the site, tap the menu, then choose Install app or Add to Home screen.
